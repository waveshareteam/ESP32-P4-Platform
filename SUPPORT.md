# Support

Use the channel that best matches your request:

- For bugs in this repository, open a GitHub issue and include the board,
  example path, framework version, logs, and hardware setup.
- For feature requests or documentation improvements, open a GitHub issue or
  pull request.
- For order-specific or hardware warranty support, contact Waveshare support:
  <https://service.waveshare.com>
- For general ESP-IDF questions, see Espressif documentation and community
  forums.

Security reports should follow [SECURITY.md](SECURITY.md), not public issues.
